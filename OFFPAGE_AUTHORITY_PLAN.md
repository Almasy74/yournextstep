# Off-Page Authority Plan (30 days)

## Goal
Increase trusted mentions and links that improve organic rankings and citation likelihood in AI answers.

## Week 1: Linkable Assets
- Publish and maintain `/best-of/` as a central hub for high-confidence pages.
- Create one data-backed "methodology summary" page (using existing framework) suitable for citation.
- Ensure each decision page has clear source section and updated date.

## Week 2: Digital PR + Partnerships
- Outreach to 20 relevant newsletters/blogs in career, AI literacy, and personal finance.
- Offer 3 guest posts with original data framing from existing decision scorecards.
- Request inclusion in curated resource lists where content already matches intent.

## Week 3: Entity Signals
- Create/refresh organization profiles with consistent naming:
  - GitHub org/profile
  - LinkedIn company page
  - X profile
- Add `sameAs` references in structured data only when these profiles are active.

## Week 4: Measurement + Iteration
- Track referring domains, branded search impressions, and assisted conversions.
- Keep only outreach channels with response rate >10%.
- Update top 10 pages with new citations and internal links from `/best-of/`.

## KPIs
- +10 new relevant referring domains.
- +20% impressions on top 20 decision pages.
- +15% clicks from non-branded queries.
- +10% affiliate CTR on pages linked from off-page placements.
